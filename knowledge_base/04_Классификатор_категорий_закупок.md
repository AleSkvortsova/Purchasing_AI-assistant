## Правила классификации

- Категория определяется по основному результату.
- Лицензия на ПО — G05, разработка ПО — S05.
- Печать по готовому макету — S13, закупка готовой полиграфии — G11.
- При сомнении ассистент предлагает до трёх вариантов.
- S15 используется только после исключения более точных категорий.
