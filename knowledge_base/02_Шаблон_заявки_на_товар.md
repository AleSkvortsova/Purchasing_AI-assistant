# Шаблон заявки на закупку товара

| Поле | Требование | Пример |
|---|---|---|
| Краткое название | 5–7 слов | Ноутбуки для отдела продаж |
| Категория | По классификатору | IT-оборудование |
| Инициатор | ФИО и контакт | Анна Петрова |
| Подразделение | Подразделение-заказчик | Продажи |
| Обоснование | Зачем нужен товар | Оснащение новых сотрудников |
| Наименование | Что требуется | Ноутбук |
| Количество | Число больше нуля | 5 |
| Единица измерения | шт., комплект и т. п. | шт. |
| Характеристики | Измеримые требования | RAM от 16 ГБ |
| Требуемая дата | Дата получения | 15.09.2026 |
| Место поставки | Адрес или площадка | Центральный офис |
| Бюджет | Общая сумма | 500 000 руб. |
| Бюджетный статус | Бюджетная / внебюджетная | Бюджетная |
| Эквивалент | Да / нет с обоснованием | Да |
| Контакт | По предметным вопросам | Анна Петрова |

## Дополнительные поля

- бренд или референс;
- совместимость;
- упаковка и комплектность;
- гарантия;
- сертификаты;
- график поставки;
- спецификация;
- фото или чертёж.

## Правила

1. Несколько позиций объединяются, только если относятся к одной категории.
2. Указывается общая сумма заявки.
3. Ссылка не заменяет характеристики.
4. «Как в прошлый раз» допустимо только со ссылкой на прежнюю заявку.
5. Дата не может быть в прошлом.
6. Количество должно быть больше нуля.
