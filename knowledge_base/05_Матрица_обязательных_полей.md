# Матрица обязательных полей

## Общие поля

Для всех заявок обязательны: название, тип, категория, инициатор, подразделение, обоснование, требуемая дата, бюджет, бюджетный статус и контактное лицо.

## Формальные проверки

- дата не в прошлом;
- количество больше нуля;
- сумма неотрицательная;
- единица измерения указана;
- окончание не раньше начала;
- сумма заявки согласована с позициями;
- при запрете эквивалента есть обоснование;
- при срочности есть причина;
- при внебюджетной закупке указан статус согласования.
