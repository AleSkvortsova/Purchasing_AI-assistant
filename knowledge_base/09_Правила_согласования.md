# Правила согласования заявок

## Матрица согласования

| Условие | Согласующие |
|---|---|
| Бюджетная закупка до 100 000 руб. | Руководитель подразделения |
| Бюджетная закупка 100 001–500 000 руб. | Руководитель + финансовый контролёр |
| Бюджетная закупка свыше 500 000 руб. | Руководитель + финансы + руководитель закупок |
| Внебюджетная до 100 000 руб. | Руководитель + финансовый директор |
| Внебюджетная свыше 100 000 руб. | Руководитель + финансовый директор + генеральный директор |
| Срочная P1/P2 | Руководитель подразделения + руководитель закупок |
| Единственный поставщик | Руководитель + руководитель закупок; свыше 500 000 руб. также финансы |
| IT с доступом к данным | IT / информационная безопасность |
| Юридические услуги | Юридическая служба |
| Работы на объекте | АХО / охрана труда |

Ассистент может определить предварительный маршрут и проверить наличие обоснований, но не может одобрять заявку или считать согласование полученным со слов пользователя.

Рекомендуемый срок ответа одного согласующего — один рабочий день.
