# Статусы заявки и переходы

| Статус | Значение | Кто устанавливает |
|---|---|---|
| Черновик | Заявка заполняется | Ассистент / инициатор |
| Передана в отдел закупок | Подтверждена и зарегистрирована | Ассистент |
| Требует доработки | Нужны сведения заказчика | Закупщик |
| Принята в работу | Полнота подтверждена | Закупщик |
| На согласовании | Требуется решение | Закупщик / система |
| В работе | Проводится закупочная процедура | Закупщик |
| Поставка ожидается | Поставщик выбран | Закупщик |
| Выполнена | Товар или услуга получены | Закупщик |
| Отклонена | Закупка не согласована | Согласующий |
| Отменена заказчиком | Потребность отменена | Инициатор / закупщик |

## Допустимые переходы

```text
Черновик
  → Передана в отдел закупок
  → Отменена заказчиком

Передана в отдел закупок
  → Требует доработки
  → Принята в работу
  → На согласовании
  → Отклонена
  → Отменена заказчиком

Требует доработки
  → Передана в отдел закупок
  → Отменена заказчиком

Принята в работу
  → На согласовании
  → В работе
  → Отклонена

На согласовании
  → Принята в работу
  → В работе
  → Отклонена

В работе
  → Поставка ожидается
  → Отклонена

Поставка ожидается
  → Выполнена
```

Каждый переход сохраняется с датой, автором и комментарием.
