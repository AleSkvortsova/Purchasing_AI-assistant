# База знаний ИИ-ассистента по закупкам

Учебная база знаний для синтетической компании ООО «Вектор Дистрибуция».

- Версия: 1.0
- Дата: 21.07.2026
- Владелец: руководитель отдела закупок
- Периодичность проверки: ежеквартально

Все сведения, суммы, сроки, роли и примеры являются синтетическими.

## Приоритет документов

1. Регламент подачи и обработки заявок.
2. Матрица обязательных полей.
3. Классификатор категорий и справочник статусов.
4. FAQ и примеры.

Ассистент не должен придумывать правила. При недостатке информации он задаёт уточняющий вопрос или направляет пользователя в отдел закупок.
