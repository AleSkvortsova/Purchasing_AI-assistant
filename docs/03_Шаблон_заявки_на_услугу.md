## Дополнительные поля

- этапность;
- формат отчётности;
- конфиденциальность;
- необходимость выезда;
- права на результат;
- SLA и сервисные показатели;
- исходные материалы.

## Правила

1. Результат должен быть проверяемым.
2. Критерии приёмки не должны быть только субъективными.
3. Указываются зависимости от исходных данных заказчика.
4. Для работ на территории компании указываются режим доступа и безопасность.
5. Для услуг с интеллектуальными правами условия передачи прав описываются отдельно.
